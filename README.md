# Campus Navigation Assistant

## Problem
Freshers and visitors often struggle to navigate large campuses because internal paths are not available on common map applications.

## Solution
This project provides a navigation-based campus assistant that gives step-by-step, landmark-based directions instead of only computing shortest paths.

## Features
- Turn-by-turn navigation instructions
- Landmark-based guidance
- Accessibility-friendly routes
- Frontend-only implementation

## Tech Stack
- HTML
- CSS
- JavaScript

## Key Learnings
- Graph-based navigation modeling
- User-centric design thinking
- Accessibility-aware routing
